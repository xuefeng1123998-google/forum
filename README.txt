
# 大学社团论坛 · 轻量可用版

这是一个最小可用版本，适合先跑通流程：
- 注册 / 登录
- 发帖
- 评论
- 点赞
- 公告置顶
- 精华帖
- 管理员后审下架

## 运行方式

1. 先安装 Python 3
2. 安装依赖：
   pip install flask werkzeug
3. 进入项目目录后运行：
   python app.py
4. 浏览器打开：
   http://127.0.0.1:5000

## 演示账号
- 管理员：admin / admin123
- 成员：member1 / member123
